## Dimension Scores

### A. Ecosystem Health (25%) — Score: 7.0

Growing adoption driven by integrations with major agent frameworks (MCP, OpenClaw, Hermes). Community is still early but has strong signal quality — developers embedding Hindsight as infrastructure rather than just experimenting. Integration count is the key metric to track here, not raw stars. The "embedded by default" pattern suggests durable ecosystem positioning.

### B. Team & Governance (20%) — Score: 7.5

Chris Latimer brings enterprise-grade background (Google Cloud Solution Architecture → DataStax PM → Vectorize). The team of 7 generating $1M ARR demonstrates exceptional capital and execution efficiency. Lean team is both a strength (efficiency) and a risk (bus factor, scaling capacity). FY16 Sales Engineer of the Year at Apigee suggests strong GTM instincts.

### C. Technical Moat (20%) — Score: 8.0

SOTA benchmark performance on agent memory retrieval, independently validated by Virginia Tech — this is unusually strong for an early-stage project. The memory layer sits at a critical position in the agent stack: every agent needs persistent memory, and whoever becomes the default wins distribution. Multi-framework compatibility (MCP, OpenClaw, Hermes) creates switching costs.

### D. Product-Market Fit (20%) — Score: 7.5

$1M ARR from 7 people is strong PMF signal. The agent memory market is structurally growing — every new agent framework release expands addressable market. Being "embedded by default" in multiple frameworks creates a data flywheel: more users → more memory patterns → better retrieval → more integrations.

### E. Exit Potential (15%) — Score: 7.5

Natural M&A target for agent platform companies (LangChain, OpenAI, Anthropic) or database companies (MongoDB, Datastax, Pinecone) wanting to own the agent memory layer. Revenue traction at this team size makes it attractive even for acqui-hire. IPO path requires significant scale-up.
